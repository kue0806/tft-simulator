
## Setup

### Prerequisites
- Python 3.11+
- Conda (Miniconda or Anaconda)

### Installation

1. Create and activate the conda environment:
```bash
conda activate tft-sim
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```
